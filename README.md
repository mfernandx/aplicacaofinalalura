# 💰 Simulador de Conta Bancária

## 📖 Sobre o projeto

Este projeto foi desenvolvido em **Java** como parte dos meus estudos iniciais na linguagem. A aplicação simula o funcionamento básico de uma conta bancária através do terminal, permitindo ao usuário consultar saldo, realizar depósitos e efetuar transferências.

O objetivo principal deste projeto foi praticar conceitos fundamentais de programação, como variáveis, estruturas condicionais, laços de repetição e interação com o usuário utilizando a classe `Scanner`.

---

## 🚀 Funcionalidades

✔️ Exibição dos dados do cliente  
✔️ Consulta de saldo da conta  
✔️ Recebimento de valores (depósitos)  
✔️ Realização de transferências  
✔️ Validação de saldo insuficiente  
✔️ Menu interativo no terminal
